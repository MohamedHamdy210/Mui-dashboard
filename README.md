# 📊 Mui Dashboard

A React + Vite dashboard app built with MUI (Material UI) for UI components.  
Live version: [Dashboard Demo](https://mui-dashboard-nine.vercel.app/)

---

## ⚙️ Tech Stack

- React  
- Vite  
- MUI (Material UI)  
- JavaScript  
- CSS


---

## ✨ Features

- 📅 Calendar view   
- Theming settings 
- Navigation sidebar
- Clean visuals, consistent UI components  

---

## 🛠️ Setup & Installation

```bash
# Clone the repo
git clone https://github.com/MohamedHamdy210/Mui-dashboard.git

# Install dependencies
npm install

npm run dev
# Run locally
npm run dev
